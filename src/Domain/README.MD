# Camada de Dominio

1. Common
Classes e interfaces reutilizáveis

2. Entity
Objetos/Classes que representam o "core" da aplicação juntamente com regra de negócio.

3. Enum
Define um conjunto fixo de valores, podendo representar status ou tipo dentro do dominio. 

4. Exceptions
Exceções de dominio usado para validações.

5. Validation
Classes que realizam as validações necessárias para as entidades.
